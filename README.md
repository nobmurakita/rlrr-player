## Shortcut Keys

| Key         | Function              |
|-------------|-----------------------|
| `Space`     | Play or Pause         |
| `Left`      | Rewind 5 seconds      |
| `Right`     | Forward 5 seconds     |
| `Backspace` | Move to the beginning |
| `F`         | Enter fullscreen mode |
| `ESC`       | Exit fullscreen mode  |
